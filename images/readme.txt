Images Folder for Akko POI Map
================================

This folder contains POI images that are displayed in the map modals.

Current active images:
- turn0image0.jpeg (עיר עתיקה של עכו)
- turn0image1.jpeg (מנהרת הטמפלרים)  
- turn0image3.jpeg (אולמות האבירים)
- turn0image5.jpeg (החאן הטורקי)

Extra images (not currently used):
- 7.jpeg, port.jpeg, efendi-hotel.jpg

To replace images:
1. Replace any of the turn0imageX.jpeg files with new images
2. Keep the same filename format (turn0imageX.jpeg)
3. The images will automatically appear in the POI modals

Note: Images are automatically resized to 250px height in the modal display.
